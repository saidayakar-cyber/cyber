LogWatch — Windows Security Log Monitoring

Detects and summarizes key Windows Security events to surface suspicious activity quickly.

Scope

  • Failed logons (4625)
  • User create/delete (4720/4726)
  • Group membership add/remove (4728/4729)
  • Special privileges (4672)

How It Works

PowerShell queries the Security log and writes JSON to C:\LogWatch\alerts.json.
