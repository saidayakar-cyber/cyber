LogWatch — Windows Security Log Monitoring

Detects and summarizes key Windows Security events to surface suspicious activity quickly.

Scope

  • Failed logons (4625)
  • User create/delete (4720/4726)
  • Group membership add/remove (4728/4729)
  • Special privileges (4672)

How It Works

PowerShell queries the Security log and writes JSON to C:\LogWatch\alerts.json. Optionally echoes to console.

Setup

  1. Create folder C:\LogWatch
  2. Place LogWatch.ps1
  3. Trigger test events (failed login; user add/delete)
  4. Schedule via Task Scheduler (1–2 min)

Sample Output

Rule: FailedLogon (4625, High)
Message: An account failed to log on... Status: 0xC000006D

Email Alerts (Optional)

Run .\LogWatch.ps1 -SinceMinutes 5 -ToConsole to simulate recent events and verify mail delivery.

Outcomes

  • Faster initial triage
  • Account change visibility
  • JSON ready for SIEM
