Python Password Manager

Local, file‑backed vault with hashed storage and clipboard timeout.

Design

  • Entries keyed by service + login
  • Fields: URL, email/username, secret
  • Secrets hashed (bcrypt/PBKDF2)

CLI

python pm.py add -s github -u https://github.com -e you@example.com -l mygithubuser
python pm.py get -s github -l mygithubuser --clip-timeout 10
python pm.py list

Implementation Notes

  • Validate inputs and avoid overwrites
  • Don’t log secrets
  • Unit test core crypto and clipboard code

Outcomes

  • No plaintext storage
  • Reduced copy‑paste errors
