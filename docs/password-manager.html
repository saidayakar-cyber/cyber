Python Password Manager

A local, file‑backed password manager that stores secrets safely and supports a clipboard timeout for quick paste‑then‑erase workflows.

1. Design Goals

  • No plaintext secrets at rest
  • Reproducible CLI for add/get/list flows
  • Clipboard timeout to reduce exposure

2. Data Model

Entries are identified by a service and login. Fields include URL, email/username, and the secret. The vault is serialized to disk with safe defaults.

3. Crypto & Handling

  • Use a strong KDF and hashing (e.g., bcrypt/PBKDF2) for secrets
  • Mask terminal output by default
  • Offer a --clip-timeout option to copy then auto‑clear the clipboard

4. CLI Usage

python pm.py add -s github -u https://github.com -e you@example.com -l mygithubuser
python pm.py get -s github -l mygithubuser --clip-timeout 10
python pm.py list

5. Implementation Notes

  • Validate input and avoid accidental overwrites
  • Log only non‑sensitive metadata
  • Unit‑test core crypto and clipboard routines

6. Outcomes

  • Eliminated plaintext storage by hashing secrets and masking output
  • Reduced copy‑paste errors via CLI flags and clear prompts

7. Next Steps

  • Multi‑vault support and export/import
  • OS keychain integration
  • YubiKey/TOTP helpers
