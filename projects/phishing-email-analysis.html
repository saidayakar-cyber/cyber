Phishing Email Analysis

Investigated suspicious emails via headers/URLs and documented a short response playbook.

Environment & Tools

Outlook/Gmail header tools • URL sandbox • VT hashes (lab)

How I built it

  1. Parse SPF/DKIM/DMARC fields
  2. Check URLs in sandbox/reputation lists
  3. Create a decision tree for users

How to run

  1. Export full headers and paste into analyzer
  2. Open suspicious URLs in an isolated VM only

Outcomes

  • Consistent triage in ~5 minutes
  • Reusable mini‑playbook for end‑users
