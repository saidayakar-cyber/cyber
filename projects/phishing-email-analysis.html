Phishing Email Analysis

Investigated suspicious emails via headers, URLs, and attachments; documented a short response playbook.

  • Environment & Tools
  • How I built it
  • How to run
  • Outcomes
  • Repository

Environment & Tools

Outlook/Gmail message headers, urlscan-style analysis, VirusTotal for hashes (lab)

How I built it

  1. Pulled full headers and parsed SPF/DKIM/DMARC fields
  2. Checked URLs against sandboxes and reputation lists
  3. Drafted a decision tree for quick user guidance

How to run

  1. Export full headers and paste into header analyzer
  2. Open suspicious URLs in an isolated VM only

Outcomes

  • Consistent triage outcome (benign vs. phishing) in ~5 minutes
  • Reusable mini-playbook to standardize responses for end users
