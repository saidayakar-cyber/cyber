Secure Network Lab

Segmented lab with VLANs and ACLs to restrict lateral movement; verification commands and IP plan included.

Environment & Tools

Switching emulator • Windows/Linux endpoints

How I built it

  1. Plan VLAN/IP schema and port modes
  2. Apply ACLs to limit inter‑VLAN access
  3. Validate with ping/trace and scans

How to run

  1. show vlan brief
  2. show access-lists
  3. ping 10.10.20.10

Outcomes

  • Segmentation reduced broadcast domain
  • Verification commands improved troubleshooting
