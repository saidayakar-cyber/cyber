Python Password Manager

Local password vault using Python with hashed storage and safe clipboard timeout for secrets.

  • Environment & Tools
  • How I built it
  • How to run
  • Outcomes
  • Repository

Environment & Tools

Python 3.11, hashlib/bcrypt, argparse; tested on Windows 11

How I built it

  1. Modeled secure fields and vault structure
  2. Implemented add/get/list flows and clipboard timeout
  3. Wrote basic CLI help and README

How to run

  1. python pm.py add -s github -u https://github.com -e you@example.com -l mygithubuser
  2. python pm.py get -s github -l mygithubuser --clip-timeout 10

Outcomes

  • Eliminated plaintext storage by hashing secrets and masking output
  • Reduced copy-paste errors via CLI flags and clear prompts
