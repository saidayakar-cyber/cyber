Python Password Manager

Local password vault with hashed storage and safe clipboard timeout.

Environment & Tools

Python 3.11, bcrypt/PBKDF2, argparse • Windows 11

How I built it

  1. Model secure fields & vault structure
  2. Implement add/get/list flows
  3. Write CLI help & README

How to run

  1. python pm.py add -s github -u https://github.com -e you@example.com -l mygithubuser
  2. python pm.py get -s github -l mygithubuser --clip-timeout 10

Outcomes

  • No plaintext secrets at rest
  • Reduced copy‑paste errors via CLI flags
