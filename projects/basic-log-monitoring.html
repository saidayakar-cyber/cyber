Basic Log Monitoring

Collected Windows/Linux security logs and built a quick triage checklist to identify failed logons, privilege events, and user changes.

Environment & Tools

Windows 10/11 & Ubuntu • Event Viewer / journalctl • simple syslog

How I built it

  1. Define target Event IDs & filters
  2. Export & normalize fields
  3. Draft triage checklist

How to run

  1. Event Viewer → Security log
  2. journalctl -u ssh --since '2 hours ago'

Outcomes

  • Reduced initial triage time in lab
  • Repeatable checklist for faster validation
